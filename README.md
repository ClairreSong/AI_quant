# AI_quant - 中芯国际交易数据可视化看板

基于 Tushare 数据接口，获取中芯国际 A股（688981.SH）和港股（00981.HK）近一年日线交易数据，并通过 ECharts 构建交互式 K线看板。

## 项目内容

| 文件 | 说明 |
|---|---|
| `smic_dashboard.html` | 交互式K线看板（可直接浏览器打开） |
| `data/smic_688981_SH.csv` | A股日线数据 |
| `data/smic_00981_HK.csv` | 港股日线数据 |
| `generate_smic_dashboard.py` | 看板生成脚本 |
| `save_hk_cache.py` | 港股数据缓存工具 |

## 看板功能

- **Tab切换** A股(科创板) / 港股(联交所)
- **K线图** 含年内最高/最低标记线，hover显示开高低收+涨跌幅+成交量
- **成交量柱状图** 红涨绿跌（A股惯例），hover显示成交额
- **6个统计卡片** 最新收盘价、年内高低、日均成交量、区间涨跌幅、振幅
- **DataZoom** 支持拖拽缩放时间范围

## 数据来源

- [Tushare Pro](https://tushare.pro/) - 金融数据接口
- 数据区间：2025-07-04 ~ 2026-07-03
